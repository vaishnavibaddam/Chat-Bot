# Document Question Answering Bot

A RAG-based application that allows users to ask questions about PDF documents.

## Features
- PDF text extraction
- Text chunking
- Gemini-powered question answering
- Streamlit interface

## Installation

pip install -r requirements.txt

## Run

cd src
streamlit run app.py

## Example Questions

- What skills does Vaishnavi have?
- What projects are mentioned?
- What is the CGPA?
